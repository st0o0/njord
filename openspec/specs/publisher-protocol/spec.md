# publisher-protocol Specification

## Purpose

Deprecated — the Tell-based `RegisterPublisher`/`PublishStateResult` protocol has been replaced by the streams-based `EgressEvent` MergeHub/BroadcastHub in the `egress-event` spec. Producers attach via `ISinkRef<EgressEvent>`, consumers via `ISourceRef<EgressEvent>`.

## Requirements

_All requirements removed. See `egress-event` spec for the replacement._
